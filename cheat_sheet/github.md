## lssues
```
여러가지 이슈들(문제점 등)을 의논하기 위한 것
  Assignees : 해결할 만한 담당자를 지정
  label : 무엇과 관련된 이슈인지 지정
```
## pull requests
```
fork한 repository를 수정작업하고 수정된 내용을 원본 repository에 적용시키는 것
진행과정 : 팀장 레포지토리 생성 -> 팀원 fork -> 팀원 수정 -> 팀원 Pull request -> 팀장 merge
```
## milestone
```
진행상황을 관리
```
## wiki
```

```
